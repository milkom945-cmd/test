# Генератор идей для СММ (банная ниша)

Простой сайт (без сборщиков) для генерации тем/углов/хуков и заготовок контента под банную нишу.

## Запуск

- Откройте файл `index.html` двойным кликом в браузере.
- Работает офлайн.

## Что умеет

- Фильтры: **цель**, **формат**, **сезон/повод**, **тон**, **уровень аудитории**
- Генерация идеи: тема + угол + хук + структура + кадры/сцены + CTA
- **Сюрприз‑режим**: чаще подмешивает необычные сочетания
- **Копирование** результата в буфер обмена
- **Избранное** (сохранение в LocalStorage браузера)

## Файлы

- `index.html` — разметка страницы
- `styles.css` — стили
- `app.js` — логика генератора + база идей
